# SMS Virtual Hub 📱

مشروع بوت توليد أرقام أمريكية وهمية واستقبال رسائل التحقق (SMS) بواجهة عصرية واحترافية.

## المميزات
- **توليد أرقام عشوائية**: بضغطة زر يمكنك الحصول على رقم أمريكي جديد.
- **واجهة عصرية**: تصميم Premium مظلم مع تأثيرات Glassmorphism.
- **تحديث تلقائي**: صندوق الوارد يتحدث تلقائياً عند وصول رسائل جديدة.
- **متوافق مع الهواتف**: واجهة مرنة تعمل على كافة أحجام الشاشات.

## كيفية الرفع على GitHub Pages
1. قم بإنشاء مستودع (Repository) جديد في حسابك على GitHub.
2. افتح الـ Terminal في مجلد المشروع وقم بتنفيذ الأوامر التالية:
   ```bash
   git init
   git add .
   git commit -m "First deploy"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO_NAME.git
   git push -u origin main
   ```
3. اذهب إلى إعدادات المستودع (Settings) -> Pages.
4. اختر Branch: **main** واضغط Save.
5. سيكون موقعك متاحاً خلال دقيقة على الرابط المعطى.

## ملاحظة برمجية
هذا المشروع حالياً يعمل بنظام **المحاكاة (Simulation)**. لتوصيله بخدمة حقيقية، يمكنك ربطه بـ API لخدمات مثل Twilio أو Sms-Activate عبر تعديل ملف `script.js`.
